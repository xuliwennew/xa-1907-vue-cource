## 项目名称
vue-ssr-demo

## 项目介绍
该项目是为了系统学习```Vue SSR```创建，共分为5个目录：

- ```01/``` 纯浏览器端渲染
- ```02/``` 服务端渲染，不包含Ajax初始化数据
- ```03/``` 服务端渲染，包含Ajax初始化数据
- ```04/``` 服务端渲染，使用serverBundle和clientManifest进行优化
- ```05/``` 一个完整的基于Vue + VueRouter + Vuex的SSR工程

每个目录对应学习过程中的一个阶段，循序渐进，真正从零开始搭建一个完整的```SSR```工程。
